## 📋 SOBRE O PROJETO

Landing page oficial do **URSA Embedded Bootcamp** - a formação prática em sistemas embarcados que ensina você a construir projetos reais do zero, como o **Sentinel-X** (sistema de emergência com ESP32).

Este repositório contém o código fonte da página de vendas do curso.

## 🎯 OBJETIVO

Converter visitantes em alunos através de um design moderno, responsivo e estratégico.

## 🛠️ TECNOLOGIAS

- HTML5 semântico
- CSS3 (Flexbox, Grid, Animações)
- JavaScript (ES6+)
- Particles.js
- Font Awesome
- Google Fonts
